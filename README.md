# Degenerates with Dice Website

This repository contains a simple static website for the "Degenerates with Dice" tabletop role-playing podcast.

## Getting Started

Open `index.html` in a browser to view the site. You can also serve the files using any static web server.

## Structure

- `index.html` - Main webpage with information about the podcast.
- `style.css` - Basic styling for the site.

Feel free to customize the content or styles to better suit your show.
